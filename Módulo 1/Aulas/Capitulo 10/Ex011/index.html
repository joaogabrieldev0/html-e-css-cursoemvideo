Download por Links

Para se criar um link instalável, deve-se utilizar o <a>, junto com os parâmetros "download="nomeDoArquivo.tipodoarquivo" e "type="application/zip", por exemplo, para arquivos .zip". Abaixo há uma tabela com as corretas utilizações.

Por exemplo:

Link para baixar o primeiro desafio do módulo: Clique Aqui

O código ficou assim: 

             
            

                <a href="desafio-mensagens.pdf" download="desafio-mensagens.pdf" type="application/pdf"
                >Clique Aqui </a>
